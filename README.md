## 🚀 Proyectos Destacados

### 🏪 [Variedades Angelly](https://github.com/JuanMa1015/Variedades-Angelly) — *En producción*
> Sistema POS fullstack para gestión de tienda de barrio

Sistema empresarial completo con módulos de inventario, cartera de crédito, proveedores, fidelización de clientes y auditoría de operaciones.

- **Stack:** FastAPI + PostgreSQL (Neon) + React + Tailwind CSS

- **Arquitectura:** Clean/DDD en backend, cliente HTTP centralizado en frontend
- **Testing:** 37 unit tests (pytest) + 12 escenarios BDD (Behave) — cobertura >80%
- **Actualmente en uso diario por el cliente**

### 😴 [A.S.A.P. — Sleep Apnea Analytics](https://github.com/asap-sleep-analytics) — *En desarrollo*
> Detección pasiva de apnea del sueño mediante análisis de audio y SpO2

Aplicación móvil de salud que analiza el audio del micrófono del smartphone y datos de saturación de oxígeno para detección no invasiva de apnea del sueño.

- **ML Pipeline:** Ensemble dual (Random Forest + Gradient Boosting) — **AUC 0.837**
- **Features:** MFCCs, ZCR, RMS + caídas de SpO2
- **Arquitectura:** Multi-repo — backend Python · mobile TypeScript/React Native · landing independiente
- **Umbrales clínicos diferenciados:** modo *screening* (alta sensibilidad) y modo *seguimiento* (CPAP)
